Interface StorageDelegate

Storage Delegate Interface

Put simply, this is the class that reads and writes from the database on behalf of Auth. Means you could use MySQL, Mongo, flat files, whatever you want. Agnosticism for the win!

# authFetchUserByUsername( string $instance_name, string $username )

Fetches a user by their username. This function should return either an array containing:
- id: the unique identifier for this user
- username: the username we just looked up
- password: the hashed version of the users password. If it's a success, or false if there's no user by that name
